Взаимодействие с БД

Откройте Google-Таблицу, затем откройте 3й лист (sandbox), выбрав его внизу. С ним вы будете взаимодействовать через следующие примеры. Код примеров можно найти в каталоге js/examples/ проекта В терминологии библиотеки:

  • Таблица Google - База данных.
  • Лист Таблицы Google - таблица Базы данных

INSERT
INSERT INTO sand SET ( , , ) Вставить

Синтаксис JS для этой команды:

  INSERT('sandbox', [
                      ['username', user],
                      ['level', level],
                      ['say', says]
                    ]);

Параметры:

  INSERT('таблица', [
                      [ имя_колонки, [значение_1, значение_2, ..., значение_N]],
                      [ имя_колонки, [значение_1, значение_2, ..., значение_N]],
                    ]);

UPDATE
UPDATE sand SET = WHERE id = Обновить

Синтаксис JS для этой команды:

  UPDATE(data, 'sandbox', [
                            [set_column, set_value],
                          ],
                          [
                            ['id', '=', where_value],
                          ]);

Параметры:

  UPDATE('значения из БД','таблица', [блок установки значений (SET)[, [блок условия(WHERE)[)

  UPDATE('значения из БД','таблица', [
                                          [ имя_колонки, новое_значение],
                                          [ имя_колонки, новое_значение],
                                            ...
                                        [,
                                        [
                                          [имя_колонки, ВЫРАЖЕНИЕ, [значение_1, значение_2, ..., значение_N]],
                                          [имя_колонки, ВЫРАЖЕНИЕ, [значение_1, значение_2, ..., значение_N]],
                                            ...
                                      [));

Прицип работы параметров (И, ИЛИ):

  UPDATE('значения из БД','таблица', [
                                        [ имя_колонки, новое_значение]
                                     [,
                                     [
                                        [имя_колонки, ВЫРАЖЕНИЕ, [значение_1 ИЛИ значение_2 ИЛИ... ИЛИ значение_N]]
                                        И
                                        [имя_колонки, ВЫРАЖЕНИЕ, [значение_1 ИЛИ значение_2 ИЛИ... ИЛИ значение_N]]
                                        И
                                        [имя_колонки, ВЫРАЖЕНИЕ, [значение_1 ИЛИ значение_2 ИЛИ... ИЛИ значение_N]]
                                     [
